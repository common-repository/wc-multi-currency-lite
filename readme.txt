Multi Currency Lite For WooCommerce allows your customer to switch to their preferred currency and allow you to accept payments in multiple currencies.

== Description ==

- Display price switcher dropdown in menu.
- Allows your customer to switch prices in different currencies.
- Allow you to accept payments in multiple currencies.
- Update currency rates on daily basis.
- you can choose fixed manual rates or auto updated rates.
- No need to set up anything extra, it will auto convert your shipping, Taxes and everything in selected currency.

* <strong>[Get Woocommerce Multi Currency Pro:](https://wpexpertshub.com/plugins/multi-currency-pro-for-woocommerce/)</strong><br>

== Installation ==
1. Install and activate Multi Currency Lite For WooCommerce.
2. Navigate Multi Currency page(Woocommerce >> Multi Currency).
3. Api Key from [https://apilayer.com](https://apilayer.com/signup) and save settings.
4. Add currencies using "Add New Currency" button.
5. Use "Update Rates" button to update currency rates.
6. Go to Appearance >> Menus and add Currency switcher to your menu where you want to display currency switcher.
